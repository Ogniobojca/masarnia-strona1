Masarnia Jakub — Onepage

Start:
- Otwórz index.html w przeglądarce (lub wrzuć pliki na hosting).

Najważniejsze do podmiany:
- Telefon i e-mail: index.html (w 2 miejscach) + JSON-LD
- Adres domeny: canonical + og:image
- Zdjęcia galerii: assets/gallery/*.jpg

Formularz:
- Jest walidacja front-end. Aby wysyłał e-maile, podepnij backend (np. PHP/SMTP) lub usługę (Formspree/Netlify Forms).

Mapy:
- Iframe działa bez klucza API. Dla precyzyjnego pina wklej embed z Google Maps.
